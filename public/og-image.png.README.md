# OG Image Required

Create a 1200x630px PNG image for social media sharing.

## Quick Creation Options:

### Option 1: Canva (Easiest - 10 minutes)
1. Go to https://www.canva.com/create/og-images/
2. Use template: "Tech Professional Portfolio"
3. Add text:
   - "Lorenz Tazan"
   - "AI-Powered Platform Engineer"
   - "Infrastructure Automation & LLM Integration"
4. Add tech icons: Docker, Kubernetes, AWS, Python
5. Export as PNG (1200x630px)
6. Save as: `public/og-image.png`

### Option 2: Figma (15 minutes)
1. Create 1200x630px frame
2. Background: Dark gradient (slate-900 to slate-800)
3. Add your name in large font (72px)
4. Add tagline (36px)
5. Add 4-6 tech logos/icons
6. Export as PNG
7. Save as: `public/og-image.png`

### Option 3: Screenshot + Edit (5 minutes)
1. Take screenshot of your Skills page node map
2. Crop to 1200x630px
3. Add text overlay with your name + title
4. Save as: `public/og-image.png`

## Color Scheme (Match Portfolio Theme)
- Dark background: #0f172a (slate-900)
- Accent: #3b82f6 (blue-400) to #a855f7 (purple-400) gradient
- Text: #ffffff (white)
